# 📘 Markdown Syntax Guide for Docusaurus

This guide covers how to write content in `.md` or `.mdx` files for your Docusaurus project.

---

## 🏷️ Headings

```md
# H1 Title
## H2 Subtitle
### H3 Section
#### H4 Smaller Heading
```
Output:

# H1 Title
## H2 Subtitle
### H3 Section
#### H4 Smaller Heading

---

## 🧾 Paragraphs and Line Breaks

Separate paragraphs with a blank line:

```md
This is the first paragraph.

This is the second paragraph.
```
Output:

This is the first paragraph.

This is the second paragraph.

Force line break with two spaces at the end of a line:

```md
Line one␣␣  
Line two
```
Output:

Line one  
Line two

---

## ✅ Text Formatting

```md
**Bold**
*Italic*
~~Strikethrough~~
`Inline code`
```

Output:

**Bold**
*Italic*
~~Strikethrough~~
`Inline code`

## 📋 Lists

### Unordered

```md
- Item A
- Item B
  - Sub-item
```

Output:

- Item A
- Item B
  - Sub-item

### Ordered

```md
1. First item
2. Second item
```

Output:

1. First item
2. Second item

---

## 💬 Blockquotes

```md
> This is a blockquote.
```

Output:
> This is a blockquote.


---

## 💡 Code Blocks

### Inline Code

```md
Use `npm install` to install packages.
```

Output:
Use `npm install` to install packages.


### Multi-line Code Block

<pre>
\```bash
npm install
npm run start
\```
</pre>

Output:
<pre>
npm install
npm run start
</pre>

## 🎬 Embedding Videos

### ▶️ YouTube Video

```html
<iframe width="560" height="315" src="https://www.youtube.com/embed/dQw4w9WgXcQ" 
title="YouTube video" frameborder="0" allowfullscreen></iframe>
```


### 📁 Local Video

Put your video in `static/videos/intro.mp4` and use:

```html
<video controls width="100%">
  <source src="/videos/intro.mp4" type="video/mp4" />
  Your browser does not support the video tag.
</video>
```

### 🌐 External Video URL

```html
<video controls width="100%">
  <source src="https://example.com/video.mp4" type="video/mp4" />
</video>
```

Output:

<iframe width="560" height="315" src="https://www.youtube.com/embed/da6iVRSxAdE?si=AKGUPMiJW7sqRJ7R" title="YouTube video player" frameborder="0" allow="accelerometer; autoplay; clipboard-write; encrypted-media; gyroscope; picture-in-picture; web-share" referrerpolicy="strict-origin-when-cross-origin" allowfullscreen></iframe>

## 📝 Notes

- Rename `.md` files to `.mdx` if you want to use React components or HTML tags.
- Use the `static/` directory for storing images, videos, and other assets.

## 📁 `_category_.json`

Use `_category_.json` inside any `docs/` subfolder to control sidebar structure and metadata.

---

### 🧩 Basic Example

```json
{
  "label": "My Category",
  "position": 1,
  "collapsible": true,
  "collapsed": true,
  "link": {
    "type": "generated-index",
    "slug": "/my-category"
  }
}
```
