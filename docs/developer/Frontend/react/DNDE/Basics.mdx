# Drag & Drop Editor(DNDE)

A React-based MJML email editor designed for building dynamic, responsive templates using a no-code drag-and-drop interface. Built with modular components, rich inline editing, and advanced MJML processing using browser-native compilation.

## 🚀 Tech Stack

This project employs a modern frontend stack powered by **React**, **MJML**, and **Redux Toolkit**, focused on building a highly interactive, WYSIWYG email template editor.


### 🖥️ Frontend

- **React 18.1.0** – Core library for building UI components
- **React DOM 18.1.0** – Rendering library for React in the browser
- **Ant Design 4.22.7** – Rich React UI component library (e.g., Buttons, Modals, Inputs)
- **Styled Components 5.3.5** – Write scoped, component-level CSS with JS logic
- **SASS 1.52.1** – CSS preprocessor for variables, nesting, and mixins
- **React Router DOM 6.3.0** – SPA routing library for React
- **React Custom Scrollbars 2 (4.4.0)** – Customizable and performant scrollbars
- **React Color 2.19.3** – Lightweight color picker components

#### Additional Tools:
- **Hooks** – Used extensively for logic encapsulation and state/effect management
- **Context API** – Lightweight alternative to Redux for scoped state sharing (e.g., CkeditorContext, HtmlContext)


### ⚙️ State Management

- **Redux Toolkit 1.8.1** – Modern Redux setup with RTK Query built-in
- **React Redux 8.0.2** – Official bindings to connect Redux to React
- **RTK Query** – Built-in RTK data fetching and caching layer used for template API interactions


### 📦 Core Editor Libraries

- **mjml 4.12.0** – Email markup language compiler for responsive emails
- **mjml-browser 4.12.0** – Browser-compatible MJML compiler (used on frontend)
- **mjml-cli 4.12.0** – CLI-based MJML compiler (used in dev)
- **lodash 4.17.21** – Utility library for deep clone, object/array manipulation
- **html2canvas 1.4.1** – Captures HTML DOM and renders as canvas image (for preview/thumbnailing)


### 🧪 Development & Build Tools

- **TypeScript 4.6.4** – Statically typed superset of JS for maintainable code
- **Webpack 5.60.0** – Module bundler for dev/prod builds and dynamic imports
- **React Scripts 5.0.1** – CRA tooling for dev server and production build
- **Babel** (`@babel/core`, `env`, `react`, `typescript`) – Transpiler to support modern JS and JSX
- **ESLint (CRA preset)** – Linter for enforcing code quality and best practices


### 🧰 Testing

- **Jest** with **@types/jest 27.5.1** – Unit testing framework with TS support
- **React Testing Library 13.2.0** – Integration testing for React UIs
- **User Event 14.2.0** – Simulates realistic user interactions in tests
- **jest-dom 5.16.4** – Custom DOM matchers for cleaner test assertions


### 📤 Backend / Dev-only Services

- **@sendgrid/mail 7.4.6** – SendGrid’s Node.js SDK for sending test/dev emails via API

This is listed under `devDependencies` and primarily used during development for testing template output or previewing emails via actual inbox delivery.
